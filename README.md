## 🚀 Getting Started

1. <b>Pick a project folder:</b>
   - <code>cd 1-EasyTask</code>
   - <code>cd 2-InvestmentCalculator</code>
   - <code>cd 3-Components</code>
   - <code>cd 4-LifeCycle</code>
2. <b>Install dependencies:</b>
   <pre>npm install</pre>
3. <b>Run the app:</b>
   <pre>npm start</pre>
   or
   <pre>ng serve</pre>
