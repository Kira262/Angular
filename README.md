## 🚀 Getting Started

1. **Pick a project from the Project Directory above.**
2. **Open a terminal in that project folder.**
3. **Install dependencies:**
   ```cmd
   npm install
   ```
4. **Run the app:**
   ```cmd
   npm start
   ```
   or
   ```cmd
   ng serve
   ```
5. **Open your browser at:** [http://localhost:4200](http://localhost:4200)
