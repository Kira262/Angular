## 🚀 Quick Start

1. **Install dependencies:**
   ```sh
   npm install
   ```
2. **Start the backend server:**
   ```sh
   cd backend
   npm install
   node app.js
   ```
   The backend runs on [http://localhost:3000](http://localhost:3000)
3. **Run the Angular app:**
   ```sh
   cd ..
   npm start
   ```
   or
   ```sh
   ng serve
   ```
4. **Open in browser:**
   [http://localhost:4200](http://localhost:4200)
