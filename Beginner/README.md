## 🚀 Quick Start

1. **Pick a project below.**
2. **Open a terminal in that project folder.**
3. **Install dependencies:**
   ```cmd
   npm install
   ```
4. **Run the app:**
   ```cmd
   npm start
   ```
   or
   ```cmd
   ng serve
   ```
5. **Open your browser at:** [http://localhost:4200](http://localhost:4200)
