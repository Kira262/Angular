## 🚀 Quick Start

1. **Install dependencies:**
   ```sh
   npm install
   ```
2. **Run the app:**
   ```sh
   npm start
   ```
   or
   ```sh
   ng serve
   ```
3. **Open in browser:**
   [http://localhost:4200](http://localhost:4200)

- 🧩 Modular, scalable architecture
