---

## 🚀 Quick Start

1. <b>Install dependencies:</b>
   <pre>npm install</pre>
2. <b>Run the app:</b>
   <pre>npm start</pre>
   or
   <pre>ng serve</pre>
3. <b>Open in browser:</b>
   <a href="http://localhost:4200">http://localhost:4200</a>

## 🗂️ Project Structure

- <code>src/app/auth/</code> – Authentication demo

- <code>src/app/safe-link.directive.ts</code> – Safe link directive
